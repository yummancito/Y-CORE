# Tools

Herramientas externas utilizadas en el desarrollo de Y-core.

## steamless/

[Steamless](https://github.com/atom0s/Steamless) — Desempaquetador de SteamStub.

Se utiliza para analizar binarios protegidos con SteamStub durante el desarrollo y reverse-engineering de hooks. No es necesario para el funcionamiento normal de Y-core.

### Uso

```powershell
.\tools\steamless\Steamless.CLI.exe "path\to\steam_api64.dll"
```
