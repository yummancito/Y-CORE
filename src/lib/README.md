# Librerías

| Archivo | Descripción |
|---|---|
| `i18n.ts` | Sistema de internacionalización. Traducciones para todos los textos de la UI. Soporta múltiples idiomas. |
| `onlinefix-compatibility.ts` | Lista de juegos compatibles/incompatibles con Online Fix. Funciones `getCompatibility()` y `getCompatibilityReason()`. |
| `categories.ts` | Categorías de juegos para la store (acción, RPG, estrategia, etc.). |
| `recommendations.ts` | Lógica de recomendaciones de juegos basada en categorías y popularidad. |
| `y-core-api.ts` | Cliente HTTP para la API de Y-core. Funciones de auth, búsqueda, depot keys, manifests. |
